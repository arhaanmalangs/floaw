FLOAW

FLOAW is a watermark-free video editor implemented as a Python CLI over FFmpeg. This repository is not a web app, so this page is intentionally a project landing page.

Run locally

python3 floaw.py input.mp4 -o output.mp4 --overwrite

Common features

  • Trim, merge clips, resize, crop, rotate, flip
  • Speed changes, color controls, blur/sharpen, grayscale
  • Text/image overlays, audio replace/mix/mute, fades
  • No watermark added by FLOAW

See full usage in this repository’s README and run python3 floaw.py --help.
